## Was ist das?

Private Website für **tr1stan.de**.

V3 ist bewusst weniger Standard-Landingpage und mehr **Cyberdeck / HUD / Techspace**:

- Matrix-Canvas im Hintergrund
- Glitch-Headline
- HUD/Radar-Panel
- Command-Palette mit `CTRL + K`
- responsive Mobile-Menü
- Hover-Glow auf Panels
- Copy-Mail Button
- 404-Fix für alte `/pages/...` Links
- Impressum/Datenschutz als saubere Platzhalter
